## Target AWS Services

Terraform Modules from [this package](https://github.com/tedilabs/terraform-aws-ecs) were written to manage the following AWS Services with Terraform.

- **AWS ECS (Elastic Container Service)**
  - Capacity Provider
  - Cluster
  - Service
  - Task Definition
